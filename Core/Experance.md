# 大创
1. 论文一定要有逻辑  
    - 研究背景与研究现状对应自己的研究目标，同类研究中存在的问题服务于自己
    - 段内也要有逻辑，每句话之间有要逻辑
         >通过xx数据量化xxx指标，进而挖掘xx关联，从而构建xx模型，实现xxx模拟/预测，以更好地为社会相关xxx服务
2. 写报告一定要质量高
   锻炼自己写高质量报告的能力，敏锐，直观，有逻辑。一击必中，不要将废话，乱话。
